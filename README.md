# Таня Гроттер и Магический контрабас

Браузерная 3D action-adventure на Babylon.js.  
Жанр: исследование + загадки (без боёв), по первой книге цикла о Тане Гроттер.

## Геймплей

- **Полёты на контрабасе** — основная механика передвижения (WASD / стрелки, Пробел / Shift)
- **Только загадки + исследование** — никаких боёв
- **Система кармы** (–100 … +100) влияет на диалоги, отделение и три концовки
- **Отделения**: Светлое / Тёмное / Смешанное — динамически переключаются
- **Эмоции Тани** — иконка + частицы при триггерных словах (родители, жалость, Чума-дель-Торт)
- **Внутренний монолог** — Таня говорит каждые 20–40 секунд (зависит от акта и кармы)
- **Избранность** — родинка светится в ключевые моменты (акты 4–5)

## Структура (5 актов по книге)

| Акт | Сцена | Длительность |
|-----|-------|--------------|
| 1 | Пролог — Москва, квартира Дурневых | ~10–20 мин |
| 2 | Прибытие и распределение | ~20–40 мин |
| 3 | Школьная жизнь + загадки | ~3–6 ч |
| 4 | Кульминация — угроза Тибидохсу | ~1–2 ч |
| 5 | Финал — Безымянный Подвал | ~30–60 мин |

## Архитектура исходного кода

```
src/
├── main.js              — точка входа: движок, сцена, рендер-цикл
├── state.js             — глобальное состояние (карма, отделение, флаги квестов, эмоция)
├── tanya.js             — монолог Тани (пулы линий по актам, рандомные интервалы)
├── flight.js            — полёт на контрабасе (ввод, физика, частицы, свечение родинки)
├── emotions.js          — иконки эмоций + автодетектирование триггеров в тексте
├── karma.js             — применение дельты кармы + UI-уведомление
├── division.js          — визуальная тема (цвета, подпись отделения) по текущему делению
├── acts/
│   ├── index.js         — менеджер актов: запуск, переход, сброс
│   ├── act1.js          — Пролог (Москва)
│   ├── act2.js          — Прибытие + выбор клятвы
│   ├── act3.js          — Школьная жизнь (Драконбол, библиотека, пророчество)
│   ├── act4.js          — Кульминация (трещина, элементы пророчества)
│   └── act5.js          — Финал (арфа/статуя, 3 концовки)
└── puzzles/
    ├── basePuzzle.js    — базовый класс загадки (карма, подсказки, solve/fail)
    └── bassCase.js      — загадка «открыть футляр контрабаса» (Акт 1)
```

## Технологии

- **Babylon.js 7** — 3D-движок
- **Vite 5** — сборка и dev-сервер
- **ESM** — чистые ES-модули без транспиляции

## Запуск

```bash
npm install
npm run dev     # http://localhost:5173/Tg-game/
npm run build   # production build → dist/
```
